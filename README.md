
# 🚗 Auto Hive API

A secure and scalable REST API for the Auto Hive car rental application. Built with Express.js, MongoDB, and Better Auth JWT verification.



# 🔗 Live API URL

```bash
https://your-api-url.vercel.ap](https://autohive-api.vercel.app
```



# 📌 Project Purpose

This API allows users to:

- Add and manage cars
- Search and filter available cars
- Book rental cars
- Manage personal bookings
- Secure routes using JWT authentication
- Update and delete their own car listings

---

# ⚙️ Technologies Used

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Better Auth
- JOSE
- CORS
- dotenv
